## 数据格式

所有文件均为 JSON 格式，各条目含字段：
- `title` — 条目名称
- `content` — 原始文本内容
- `type` — 分类标识
- `source` — 数据来源
- `metadata` — 扩展信息

## 数据来源

- 角色/武器/材料/敌人/书籍/圣遗物：游戏内图鉴 + 游戏数据提取
- 魔神任务对话：基于 [GenshinDialog](https://github.com/mrzjy/GenshinDialog) 项目提取
- 角色档案：基于 [GenshinDialog](https://github.com/mrzjy/GenshinDialog) avatar 数据
